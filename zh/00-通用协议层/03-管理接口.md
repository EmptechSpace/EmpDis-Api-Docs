## 查询外设初始化状态（CheckInitState）

通过本条指令上层应用可以查询外设的初始化状态。

### 请求参数

请求示例：

```json
{
  "seq": "MAN_Device_CheckInitState_${uuid}",
  "cmd": "CheckInitState",
  "datetime": "20211201130101",
  "timeout": "30000",
  "param": {
    "dev_name": "00__Camera"
  }
}
```

参数说明：

| 参数名称 | 格式 | 是否必填 | 参数要求 |
|----------|------|----------|----------|
| seq | string | 是 | MAN_Device_CheckInitState_${uuid} |
| cmd | string | 是 | 固定为"CheckInitState" |
| datetime | string | 是 | 指令的下发时间，格式：YYYYMMddHHmmss |
| timeout | string | 是 | 超时时间(ms) |
| param | object | 否 | 参数对象 |
| ↳ dev_name | string | 否 | 要查询状态的设备名称，如果为空则表示查询所有的外设初始化状态；如果为"00__Camera"则表示查询工位号为00的摄像头初始化状态 |

### 返回参数

返回示例：

```json
{
  "seq": "MAN_Device_CheckInitState_${uuid}",
  "cmd": "CheckInitState",
  "datetime": "20211201130101",
  "code": "0",
  "data": {
    "init-num": "3",
    "init-states": [
      {
        "name": "00__Camera",
        "version": "1.22.345",
        "state": "ok",
        "reason": "******"
      },
      {
        "name": "01__Camera",
        "version": "1.22.345",
        "state": "err",
        "reason": "******"
      }
    ]
  }
}
```

参数说明：

| 参数名称 | 格式 | 是否必填 | 参数要求 |
|----------|------|----------|----------|
| seq | string | 是 | 同下发的 seq |
| cmd | string | 是 | 同下发的 cmd |
| datetime | string | 是 | 指令的下发时间，格式：YYYYMMddHHmmss |
| code | string | 是 | 参照通用返回码 |
| data | object | 否 | 返回的数据对象 |
| ↳ init-num | string | 否 | 外设的初始化信息节点的数量 |
| ↳ init-states | Array | 否 | 外设初始化信息节点列表 |
| ↳↳ [0].name | string | 是 | 外设名称，"工位__类型" 格式 |
| ↳↳ [0].version | string | 是 | 外设库版本 |
| ↳↳ [0].state | string | 是 | ok: 初始化成功; err: 初始化失败 |
| ↳↳ [0].reason | string | 否 | 初始化失败原因 |

## 外设健康状态检查（CheckHealthState）

在每次业务开始前，建议调用该指令对外设运行状态进行检查，以确保设备处于正常工作状态。

### 请求参数

请求示例：

```json
{
  "seq": "MAN_Device_CheckHealthState_uuid",
  "cmd": "CheckHealthState",
  "timeout": "30000"
}
```

参数说明：

| 参数名称 | 格式 | 是否必填 | 参数要求 |
|----------|------|----------|----------|
| seq | string | 是 | MAN_Device_CheckHealthState_${uuid} |
| cmd | string | 是 | 固定为"CheckHealthState" |
| datetime | string | 是 | 指令的下发时间，格式：YYYYMMddHHmmss |

### 返回参数

返回示例：

```json
{
  "seq": "MAN_Device_CheckHealthState_uuid",
  "cmd": "CheckHealthState",
  "code": "0",
  "msg": "Success",
  "health-states": [
    {
      "name": "00__Camera",
      "code": "0",
      "healthy": true,
      "reason": ""
    },
    {
      "name": "00__QrScan",
      "code": "0",
      "healthy": false,
      "reason": "serial port open error"
    }
  ]
}
```

参数说明：

| 参数名称 | 格式 | 是否必填 | 参数要求 |
|----------|------|----------|----------|
| seq | string | 是 | 同下发的 seq |
| cmd | string | 是 | 同下发的 cmd |
| code | string | 是 | 参照通用返回码 |
| msg | string | 否 | 参照通用返回码 |
| health-states | Array | 是 | 返回的健康状态列表 |
| ↳ [0].name | string | 是 | 外设名称，"工位__类型" 格式 |
| ↳ [0].code | string | 是 | HealthCheck 错误码 |
| ↳ [0].healthy | string | 是 | true: 健康; err: 不健康 |
| ↳ [0].reason | string | 否 | 初始化失败原因 |
