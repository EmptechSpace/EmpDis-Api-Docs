| 版本 | 日期 | 修改内容 |
|------|------|----------|
| V1.0 | 2026-06-16 | 初始版本，从原始文档拆分 |

## 设备信息

| 项目 | 内容 |
|------|------|
| 设备类型 | PCSC 读卡器 |
| 品牌 | 雄帝 |
| 型号 | EMP5650C |
| DIS 接口前缀 | DEV_PCSC |

## 概述

本模块提供符合 PC/SC 标准的设备连接及 APDU 通信能力，并在 Windows 系统下基于 WinSCard 接口实现。

## 调用流程

```
Open → Read → (获取读取结果) → Close
```

## 接口列表

### 1. 打开读写器（Open）

#### 请求参数

请求示例：

```json
{
  "seq": "DEV_PCSC_Open_${uuid}",
  "cmd": "Open",
  "datetime": "20211201130101",
  "posidx": "00",
  "Timeout": "30000",
  "ASYNC": "0"
}
```

参数说明：

| 参数名称 | 格式 | 是否必填 | 参数说明 |
|----------|------|----------|----------|
| seq | string | 是 | DEV_PCSC_Open_${uuid} |
| cmd | string | 是 | 固定为"Open" |
| datetime | string | 是 | 指令的下发时间，格式：YYYYMMddHHmmss |
| posidx | string | 是 | 多个同款设备的工位号；"00"~"99" |
| Timeout | string | 是 | 超时时间(ms) |
| ASYNC | string | 是 | 是否异步（建议为1）；0：同步；1：异步 |

#### 返回参数

返回示例：

```json
{
  "seq": "DEV_PCSC_Open_${uuid}",
  "cmd": "Open",
  "datetime": "20211201130102",
  "code": "0",
  "msg": "success",
  "posidx": "00"
}
```

参数说明：

| 参数名称 | 格式 | 是否必填 | 参数说明 |
|----------|------|----------|----------|
| seq | string | 是 | 同下发的 seq |
| cmd | string | 是 | 同下发的 cmd |
| datetime | string | 是 | 指令的下发时间，格式：YYYYMMddHHmmss |
| code | string | 是 | 参照通用返回码 / 读卡器返回码 |
| msg | string | 否 | 提示信息 |
| posidx | string | 是 | 多个同款设备的工位号；"00"~"99" |

### 2. 读护照（Read）

通过本条指令上层应用可以读取护照芯片信息。

#### 请求参数

请求示例：

```json
{
  "seq": "DEV_PCSC_Read_${uuid}",
  "cmd": "Read",
  "datetime": "20211201130101",
  "posidx": "00",
  "Timeout": "30000",
  "ASYNC": "1",
  "param": {
    "Birth": "890613",
    "CardTypeEnum": "8",
    "Expiry": "240422",
    "FacePath": "./face",
    "PassportNo": "E16353007"
  }
}
```

参数说明：

| 参数名称 | 格式 | 是否必填 | 参数说明 |
|----------|------|----------|----------|
| seq | string | 是 | DEV_PCSC_Read_${uuid} |
| cmd | string | 是 | 固定为"Read" |
| datetime | string | 是 | 指令的下发时间，格式：YYYYMMddHHmmss |
| posidx | string | 是 | 多个同款设备的工位号；"00"~"99" |
| Timeout | string | 是 | 超时时间(ms) |
| ASYNC | string | 是 | 是否异步（建议为1）；0：同步；1：异步 |
| param | object | 是 | 入参 |
| ↳ Birth | string | 是 | 出生日期 |
| ↳ Expiry | string | 是 | 有效期 |
| ↳ FacePath | string | 是 | 人像保存文件名路径 |
| ↳ PassportNo | string | 是 | 护照号 |
| ↳ CardTypeEnum | string | 是 | 证件类型 |

#### 返回参数

返回示例：

```json
{
  "cmd": "Read",
  "code": "0",
  "data": {
    "Birth": "890613",
    "CountryCode": "CHN",
    "Expiry": "240422",
    "FacePath": "./face.jp2",
    "Mrz1": "POCHNZENG<<LEI<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<",
    "Mrz2": "E163530073CHN8906131M2404222NEPIMANH<<<<A994",
    "Nationally": "CHN",
    "PassportNo": "E16353007",
    "Sex": "M",
    "Type": "PO",
    "detail": "CMD:51,Para=24 CtrByte=00000C03 MRZ=E163530073<<<8906131<2404222<<<<<<<<<<<<<<<<",
    "fullname": "曾 亮"
  },
  "datatime": "20260602194737.614",
  "msg": "--Success",
  "posidx": "0",
  "seq": "DEV_PCSC_Read_${uuid}"
}
```

参数说明：

| 参数名称 | 格式 | 是否必填 | 参数说明 |
|----------|------|----------|----------|
| seq | string | 是 | 同下发的 seq |
| cmd | string | 是 | 同下发的 cmd |
| datetime | string | 是 | 指令的下发时间，格式：YYYYMMddHHmmss |
| code | string | 是 | 参照通用返回码 / 读卡器返回码 |
| msg | string | 否 | 提示信息 |
| posidx | string | 是 | 多个同款设备的工位号；"00"~"99" |
| data | object | 否 | 返回数据 |
| ↳ Birth | string | 是 | 出生日期 |
| ↳ CountryCode | string | 是 | 国家代码 |
| ↳ Expiry | string | 是 | 有效期 |
| ↳ FacePath | string | 是 | 人像保存路径 |
| ↳ Mrz1 | string | 否 | MRZ 第一行 |
| ↳ Mrz2 | string | 否 | MRZ 第二行 |
| ↳ Nationally | string | 是 | 国籍 |
| ↳ PassportNo | string | 是 | 护照号 |
| ↳ Sex | string | 是 | 性别 |
| ↳ Type | string | 是 | 证件类型 |
| ↳ detail | string | 否 | 读取详情 |
| ↳ fullname | string | 是 | 姓名 |

| 序号 | 错误码 | 含义 |
|------|--------|------|
| 1 | 12666100 | BAC 验证失败 |

| 3 | 12617001 | 参数对象为空 |
| 4 | 12651203 | 打开文件失败 |
| 5 | 12651001 | 设备未打开 |
| 6 | 12662101 | 选择卡或应用失败 |
