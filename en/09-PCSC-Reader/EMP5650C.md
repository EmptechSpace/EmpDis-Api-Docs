|------|------|

| Brand | Emperor |
| Model | EMP5650C |
| DIS Interface Prefix | DEV_PCSC |

## Overview

This module provides PC/SC standard-compliant device connection and APDU communication capabilities, implemented on Windows systems based on the WinSCard interface.

## Call Flow

```
Open → Read → (Get Read Result) → Close
```

## Interface List

### 1. Open Reader (Open)

#### Request Parameters

Request Example:

```json
{
  "seq": "DEV_PCSC_Open_${uuid}",
  "cmd": "Open",
  "datetime": "20211201130101",
  "posidx": "00",
  "Timeout": "30000",
  "ASYNC": "0"
}
```

Parameter Description:

| Parameter Name | Format | Required | Description |
|----------|------|----------|----------|
| seq | string | Yes | DEV_PCSC_Open_${uuid} |
| cmd | string | Yes | Fixed as "Open" |
| datetime | string | Yes | Command dispatch time, format: YYYYMMddHHmmss |
| posidx | string | Yes | Station number for multiple devices of the same type; "00"~"99" |
| Timeout | string | Yes | Timeout (ms) |
| ASYNC | string | Yes | Async flag (recommended 1); 0: synchronous; 1: asynchronous |

#### Return Parameters

Return Example:

```json
{
  "seq": "DEV_PCSC_Open_${uuid}",
  "cmd": "Open",
  "datetime": "20211201130102",
  "code": "0",
  "msg": "success",
  "posidx": "00"
}
```

Parameter Description:

| Parameter Name | Format | Required | Description |
|----------|------|----------|----------|
| seq | string | Yes | Same as the dispatched seq |
| cmd | string | Yes | Same as the dispatched cmd |
| datetime | string | Yes | Command dispatch time, format: YYYYMMddHHmmss |
| code | string | Yes | Refer to General Return Codes / Reader Return Codes |
| msg | string | No | Prompt message |
| posidx | string | Yes | Station number for multiple devices of the same type; "00"~"99" |

### 2. Read Passport (Read)

Through this command, the upper-layer application can read passport chip information.

#### Request Parameters

Request Example:

```json
{
  "seq": "DEV_PCSC_Read_${uuid}",
  "cmd": "Read",
  "datetime": "20211201130101",
  "posidx": "00",
  "Timeout": "30000",
  "ASYNC": "1",
  "param": {
    "Birth": "890613",
    "CardTypeEnum": "8",
    "Expiry": "240422",
    "FacePath": "./face",
    "PassportNo": "E16353007"
  }
}
```

Parameter Description:

| Parameter Name | Format | Required | Description |
|----------|------|----------|----------|
| seq | string | Yes | DEV_PCSC_Read_${uuid} |
| cmd | string | Yes | Fixed as "Read" |
| datetime | string | Yes | Command dispatch time, format: YYYYMMddHHmmss |
| posidx | string | Yes | Station number for multiple devices of the same type; "00"~"99" |
| Timeout | string | Yes | Timeout (ms) |
| ASYNC | string | Yes | Async flag (recommended 1); 0: synchronous; 1: asynchronous |
| param | object | Yes | Input parameters |
| ↳ Birth | string | Yes | Date of birth |
| ↳ Expiry | string | Yes | Expiry date |
| ↳ FacePath | string | Yes | Portrait save file path |
| ↳ PassportNo | string | Yes | Passport number |
| ↳ CardTypeEnum | string | Yes | Document type |

#### Return Parameters

Return Example:

```json
{
  "cmd": "Read",
  "code": "0",
  "data": {
    "Birth": "890613",
    "CountryCode": "CHN",
    "Expiry": "240422",
    "FacePath": "./face.jp2",
    "Mrz1": "POCHNZENG<<LEI<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<",
    "Mrz2": "E163530073CHN8906131M2404222NEPIMANH<<<<A994",
    "Nationally": "CHN",
    "PassportNo": "E16353007",
    "Sex": "M",
    "Type": "PO",
    "detail": "CMD:51,Para=24 CtrByte=00000C03 MRZ=E163530073<<<8906131<2404222<<<<<<<<<<<<<<<<",
    "fullname": "曾 亮"
  },
  "datatime": "20260602194737.614",
  "msg": "--Success",
  "posidx": "0",
  "seq": "DEV_PCSC_Read_${uuid}"
}
```

Parameter Description:

| Parameter Name | Format | Required | Description |
|----------|------|----------|----------|
| seq | string | Yes | Same as the dispatched seq |
| cmd | string | Yes | Same as the dispatched cmd |
| datetime | string | Yes | Command dispatch time, format: YYYYMMddHHmmss |
| code | string | Yes | Refer to General Return Codes / Reader Return Codes |
| msg | string | No | Prompt message |
| posidx | string | Yes | Station number for multiple devices of the same type; "00"~"99" |
| data | object | No | Return data |
| ↳ Birth | string | Yes | Date of birth |
| ↳ CountryCode | string | Yes | Country code |
| ↳ Expiry | string | Yes | Expiry date |
| ↳ FacePath | string | Yes | Portrait save path |
| ↳ Mrz1 | string | No | MRZ line 1 |
| ↳ Mrz2 | string | No | MRZ line 2 |
| ↳ Nationally | string | Yes | Nationality |
| ↳ PassportNo | string | Yes | Passport number |
| ↳ Sex | string | Yes | Gender |
| ↳ Type | string | Yes | Document type |
| ↳ detail | string | No | Read details |
| ↳ fullname | string | Yes | Full name |

## Error Codes

| No. | Error Code | Meaning |
|------|--------|------|
| 1 | 12666100 | BAC verification failed |
| 2 | 12667001 | Object is null |
| 3 | 12617001 | Parameter object is null |
| 4 | 12651203 | Failed to open file |
| 5 | 12651001 | Device not opened |
| 6 | 12662101 | Failed to select card or application |
