## 调用流程

```
OpLed（操作LED灯）
GetLed（获取LED状态）
```

## 接口列表

### 1. 操作 LED 灯（OpLed）

#### 请求参数

请求示例：

```json
{
  "seq": "DEV_LightCtrl_OpLed_${uuid}",
  "cmd": "OpLed",
  "datetime": "20211201130101",
  "posidx": "00",
  "timeout": "30000",
  "async": "0",
  "param": {
    "LedNo": 1,
    "Status": 1
  }
}
```

参数说明：

| 参数名称 | 格式 | 是否必填 | 参数说明 |
|----------|------|----------|----------|
| seq | string | 是 | 请求序列号：字段格式为业务标识头+下划线+唯一号 |
| cmd | string | 是 | 本指令下，固定为"OpLed" |
| datetime | string | 是 | 指令的下发时间，格式：YYYYMMddHHmmss |
| posidx | string | 是 | 多个同款设备的工位号；"00"~"99" |
| timeout | string | 是 | 超时时间(ms) |
| async | string | 是 | 是否异步（默认0:同步）；0：同步；1：异步 |
| param | object | 是 | 请求参数 |
| ↳ LedNo | string | 是 | 灯编号 |
| ↳ Status | string | 是 | 0：灭；1：亮；2：闪烁 |
| ↳ BlinkTime | string | 是 | 闪烁间隔时间，单位 ms |
| ↳ BlinkTimes | string | 是 | 闪烁次数 |
| ↳ Brightness | string | 是 | 灯亮度 |

#### 返回参数

返回示例：

```json
{
  "seq": "DEV_LightCtrl_OpLed_${uuid}",
  "cmd": "OpLed",
  "datetime": "20211201130101",
  "code": "0",
  "msg": "success",
  "posidx": "00",
  "async": "0"
}
```

参数说明：

| 参数名称 | 格式 | 是否必填 | 参数说明 |
|----------|------|----------|----------|
| seq | string | 是 | 同下发的 seq |
| cmd | string | 是 | 同下发的 cmd |
| datetime | string | 是 | 指令的下发时间，格式：YYYYMMddHHmmss |
| code | string | 是 | 参照控制板错误码说明 |
| msg | string | 否 | 参照控制板错误码说明 |
| posidx | string | 是 | 多个同款设备的工位号；"00"~"99" |

### 2. 获取 LED 状态（GetLed）

#### 请求参数

请求示例：

```json
{
  "seq": "DEV_LightCtrl_GetLed_${uuid}",
  "cmd": "GetLed",
  "datetime": "20211201130101",
  "posidx": "00",
  "timeout": "30000",
  "async": "1",
  "param": {
    "LedNo": "1"
  }
}
```

参数说明：

| 参数名称 | 格式 | 是否必填 | 参数说明 |
|----------|------|----------|----------|
| seq | string | 是 | 请求序列号：字段格式为业务标识头+下划线+唯一号 |
| cmd | string | 是 | 本指令下，固定为"GetLed" |
| datetime | string | 是 | 指令的下发时间，格式：YYYYMMddHHmmss |
| posidx | string | 是 | 多个同款设备的工位号；"00"~"99" |
| Timeout | string | 是 | 超时时间(ms) |
| async | string | 是 | 是否异步（默认0:同步）；0：同步；1：异步 |
| param | object | 是 | 请求参数 |
| ↳ LedNo | string | 是 | 灯编号 |

#### 返回参数

返回示例：

```json
{
  "seq": "DEV_LightCtrl_GetLed_${uuid}",
  "cmd": "GetLed",
  "datetime": "20211201130101",
  "code": "0",
  "msg": "success",
  "posidx": "00",
  "async": "1",
  "data": {
    "LedNo": "1",
    "Brightness": "200",
    "Status": "1"
  }
}
```

参数说明：

| 参数名称 | 格式 | 是否必填 | 参数说明 |
|----------|------|----------|----------|
| seq | string | 是 | 同下发的 seq |
| cmd | string | 是 | 同下发的 cmd |
| datetime | string | 是 | 指令的下发时间，格式：YYYYMMddHHmmss |
| code | string | 是 | 参照控制板错误码说明 |
| msg | string | 否 | 参照控制板错误码说明 |
| posidx | string | 是 | 多个同款设备的工位号；"00"~"99" |
| data | object | 是 | 返回数据 |
| ↳ Brightness | string | 是 | 亮度 |
| ↳ Status | string | 是 | 0：灭；1：亮；2：闪烁 |
